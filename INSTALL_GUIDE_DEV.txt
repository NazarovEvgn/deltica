## ЧТО НА ФЛЕШКЕ

Вся папка deltica (без тяжелых файлов)

═══════════════════════════════════════════════════════════════════
  ШАГ 1: Установите необходимое ПО
═══════════════════════════════════════════════════════════════════

### 1. Python 3.13
https://www.python.org/downloads/
✓ Обязательно: Add Python to PATH

### 2. Node.js 20+
https://nodejs.org/
Выберите LTS версию

### 3. uv (Python package manager)
Откройте PowerShell и выполните:
powershell -c "irm https://astral.sh/uv/install.ps1 | iex"

### 4. PostgreSQL 16
https://www.postgresql.org/download/windows/
Пароль postgres: admin123 (запомните!)

═══════════════════════════════════════════════════════════════════
  ШАГ 2: Скопируйте проект с флешки
═══════════════════════════════════════════════════════════════════

Copy-Item E:\deltica C:\Projects\deltica -Recurse

═══════════════════════════════════════════════════════════════════
  ШАГ 3: Настройте базу данных
═══════════════════════════════════════════════════════════════════

1. Откройте pgAdmin
2. Создайте базу данных: deltica_db
3. Создайте пользователя: deltica_user / deltica123
4. Дайте права пользователю на базу

═══════════════════════════════════════════════════════════════════
  ШАГ 4: Настройте проект
═══════════════════════════════════════════════════════════════════

cd C:\Projects\deltica

# Создайте .env из шаблона
copy .env.example .env

# Отредактируйте .env (Блокнот)
DB_PASSWORD=deltica123
SECRET_KEY=test_secret_key_min_32_chars_123456

# Установите Python зависимости
uv sync

# Установите frontend зависимости
cd frontend
npm install
cd ..

# Примените миграции БД
uv run alembic upgrade head

# Создайте тестового админа
uv run python backend/scripts/seed_users.py

═══════════════════════════════════════════════════════════════════
  ШАГ 5: Запустите приложение
═══════════════════════════════════════════════════════════════════

# PowerShell (запускает backend + frontend)
.\start.ps1

# Или CMD
start.bat

Откройте: http://localhost:5173
Логин: admin / Пароль: admin123

═══════════════════════════════════════════════════════════════════

Готово! Deltica работает в dev режиме.

Для production версии используйте build-scripts.
