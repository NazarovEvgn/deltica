# План разработки

## Подготовка к деплою на сервер.
- Было принято решение, что приложение будет устанавливаться через exe файл, а не через docker
 - удалить из проекта подготовку для разворачивания через Docker.
- Сейчас на рабочих ПК работники входят под своими логинами/пароля через windows аутентификацию.
- Нужно внедрить в приложение аутентификацию windows, чтобы не плодить доп логины и пароли. Т.е. работник пришел, включил свой ПК, ввел  логин пароль для windows и при загрузке Deltica автоматически был узнан приложением.
- Но подхват ролей должен сохраниться.
- Наприме по логину определяется пользователь админ или лаборант.
